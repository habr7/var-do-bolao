# Comandos do VAR do Bolao

## Admin
| Comando | Descricao |
|---------|-----------|
| `!criar bolao [nome]` | Cria um bolao no grupo |
| `!abrir rodada [N]` | Abre rodada N para palpites |
| `!fechar rodada` | Fecha rodada atual |
| `!calcular` | Calcula pontuacao |
| `!pausar bolao` | Pausa o bolao |
| `!encerrar bolao` | Finaliza o bolao |

## Participante
| Comando | Descricao |
|---------|-----------|
| `!entrar` | Entra no bolao |
| `!sair` | Sai do bolao |
| `!palpite` | Inicia fluxo de palpites |
| `!meu palpite` | Mostra seus palpites |
| `!ranking` | Ranking geral |
| `!rodada` | Jogos da rodada |
| `!meus pontos` | Sua pontuacao |
| `!resultados` | Resultados dos jogos |
| `!ajuda` | Lista comandos |
